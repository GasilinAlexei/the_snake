# the_snake

Классическая игра «Змейка» — минималистичная реализация легендарной аркады.  
Игрок управляет змейкой, собирает еду, увеличивает счёт и старается избежать столкновений.

---

## О проекте

**the_snake** — это простая, лёгкая и понятная версия игры «Змейка», созданная для обучения и демонстрации базовой игровой логики.

В проекте можно увидеть:
- реализацию движения змейки;
- генерацию еды;
- обработку столкновений;
- игровую петлю;
- отображение состояния игры.

---

##  Технологии

- **Python**
-  библиотека для рендеринга `pygame`  
---

## Установка и запуск

### 1. Клонируйте репозиторий:

```bash
git clone https://github.com/GasilinAlexei/the_snake.git
cd the_snake
```

### 2. Создайте виртуальное окружение:

```bash
python3 -m venv env
source env/bin/activate      # Linux/macOS
env\Scripts\activate         # Windows
```

### 3. Установите зависимости:

```bash
pip install -r requirements.txt
```

### 4. Запустите игру:

```bash
python the_snake.py
```
